# Tidy Data Code Book
## Coursera - Cleaning Data Course - Final Project

This code book explains the tidy data resulting from the processing of the Human Activity Recognition Using Smartphones data set.

The data can be found in the tidyDataNarrow.txt tab delimitted file.


## Data Columns

**subjectId**
*integer*
The person/subject identifier who participated in the data collection
Possilble Values: 1-30

**activityId**
*integer*
The activity identifer indicating which activity the measurement is for 
Possible Values: 1-6

**activityName**
*string*
The name of the activity that the measurement corresponds to 
Possible Values: laying, sitting, standing, walking, walking_downstairs, walking_upstairs
